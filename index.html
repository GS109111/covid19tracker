What Can The Tracker Do?

  • Display Current Covid Cases Using Your Current Location or Custom Location
  • Graph The Cases Using Matplotlib
  • Easy To Use Interface and Provides Latest & Accurate Cases

How To Install & Run The Tracker (On MacOS)?

  1. Download The Tracker
  2. Make Sure You Have Python3 Installed
  3. Open Terminal From Applications > Utilites
  4. Enter cd "drag path of covid 19 tracker" into the prompt
  5. Enter sh mac_installer.sh into the prompt
  6. To Run the Project: Enter "python3 main.py" into the prompt

How To Install & Run The Tracker (On Windows)?

  1. Download The Tracker
  2. Make Sure You Have Python3 Installed
  3. Open Command Prompt From The Start Menu
  4. Drag "windows_installer.bat" into the prompt
  5. To Run the Project: Enter "python main.py" into the prompt
